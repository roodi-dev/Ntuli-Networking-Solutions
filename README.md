# Ntuli Networking Solutions — Website

Local preview:

Open `index.html` in a browser, or serve the folder with a simple static server (for example `npx serve` or the Live Server extension).

Site structure:

- `index.html` — homepage with overview and quick links
- `about.html` — company and team information
- `services.html` — service details and imagery
- `contact.html` — contact form and office info
- `css/styles.css` — shared styling
- `js/script.js` — form helper, year, and mobile nav toggle

Images are referenced from Unsplash (free, rights-managed images) using keyword-based source URLs. Replace these with local images in the `images/` folder if you prefer.

To deploy: copy files to any static host (GitHub Pages, Netlify, Vercel, or your web server).
